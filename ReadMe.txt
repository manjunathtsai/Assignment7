FORMULA 

jin = jout * s

rout = rin + (k – 1) * jin

jin = jump
s = stride 
rin = receptive field input
rout = receptive field output
k = kernal size

====================================================================
Layer 1 

jout = 1 , s = 2 , rin = 1 . k =7

jin = 1 x 2
    = 2 

rout = 1 + (7 – 1) * 2
     = 13
